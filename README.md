# Unit-Value

🚀 First SwiftUI Challenge Complete!

I just built my first iOS app from scratch - a Unit Converter that handles length conversions between Meters, Kilometers, and Feet.

📱 What the app does:
• Users can select input AND output units
• Enter any value
• Instantly see the converted result

🛠 What I learned:
• SwiftUI basics (Forms, NavigationStack, Pickers)
• @State and @FocusState property wrappers
• Keyboard management with Toolbar
• Unit conversion logic and math operations
• Handling user input with TextField

This challenge taught me how to think like an iOS developer - breaking down problems, handling edge cases, and creating intuitive user interfaces.
